# Manual DOCENTE

## 1. Ingreso
1. Abrir panel ADMIN o vista asignada por institucion.
2. Iniciar sesion con cuenta DOCENTE.

## 2. Alcance del docente
- Solo ve colegios/salones/estudiantes de su alcance asignado por ADMIN.
- No accede a Operacion del sistema ni a funciones administrativas globales.

## 3. Uso diario
- Revisar dashboard docente.
- Consultar resultados permitidos.
- Aplicar filtros por prueba, nivel, materia y estudiante.
- Identificar estudiantes en riesgo academico.

## 4. Si faltan estudiantes
1. Verificar filtros activos.
2. Confirmar con ADMIN el colegio/salon asignado.
3. Solicitar validacion de alcance si persiste.

## 5. Si observa datos incorrectos
- Reportar numero de documento del estudiante, prueba y hora del evento al ADMIN.

## 6. Restricciones
- No puede gestionar usuarios administrativos.
- No puede ejecutar importaciones, backup o preparacion local.
