# Manual ESTUDIANTE

## 1. Ingreso
1. Abrir `http://192.168.1.25:4000/simulador/`.
2. Iniciar sesion con tipo y numero de identificacion.

## 2. Antes de iniciar
- Verificar nombre de prueba, nivel y numero de preguntas.
- Pulsar "Iniciar prueba".

## 3. Durante la prueba
- Leer pregunta y seleccionar opcion.
- Usar Anterior/Siguiente para navegar.
- Verificar estado de guardado: Guardando, Guardado o Pendiente.

## 4. Si se cae la conexion
- El sistema guarda respaldo temporal en navegador.
- Al volver la conexion, sincroniza respuestas automaticamente.

## 5. Si recarga o cierra navegador
- Volver a iniciar sesion.
- Continuar intento activo cuando el sistema lo indique.

## 6. Si cambia de equipo
1. Iniciar sesion en el nuevo equipo.
2. Seleccionar continuar intento activo.
3. Verificar respuestas restauradas.

## 7. Finalizar
- Confirmar envio final.
- Revisar resultado propio.
- No compartir credenciales.
